WebVR polyfill Amblyopia Letter Game

See vrlazyeye.com for more information: vrlazyeye.com

Game Control
Press "Space Bar" or "PS4 Controller L1/L2" when the numbers match. 100 points
Press "L" or left mouse-click or PS4 controller R1/R2 if numbers don't match. 10 points

To Play = press "p" and then press "f" to enter VR Mode

Lazy Eye Amblyopia Control
Weak Left eye = press "1" key
Weak Right eye = press "2" key

Use ony Firefox with PS4 Controller connected via USB to PC

Score in red box is the average Z score, which coorelates to the size of the number
for correct responses. Higher score is better = smaller letter recognition. This value,
not the "points", may be used to indicate Amblyopia progress if tracked over many sessions.
